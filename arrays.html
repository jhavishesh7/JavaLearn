Arrays

let numbers = [1, 2, 3];
numbers.push(4);
numbers.unshift(0);
let last = numbers.pop();
let first = numbers.shift();

let doubled = numbers.map(n => n * 2);
let evens = numbers.filter(n => n % 2 === 0);
let sum = numbers.reduce((total, n) => total + n, 0);

let output = `Original: ${numbers}
After operations: ${numbers}
Doubled: ${doubled}
Evens: ${evens}
Sum: ${sum}`;

document.getElementById('output').textContent = output;

Explanation

  • let numbers = [1, 2, 3]; — Creates an array of numbers.
  • push(4) — Adds 4 to the end of the array.
  • unshift(0) — Adds 0 to the beginning of the array.
  • pop() — Removes the last element (4) from the array.
  • shift() — Removes the first element (0) from the array.
  • map(n => n * 2) — Creates a new array with each value doubled.
  • filter(n => n % 2 === 0) — Creates a new array with only even numbers.
  • reduce((total, n) => total + n, 0) — Sums all numbers in the array.
  • The output variable collects the results and is displayed in the green output box above.
