Variables & Data Types

let name = "Alice";
let age = 25;
let isStudent = true;
let nothing = null;
let notDefined;
let bigNumber = 12345678901234567890n;
let sym = Symbol('unique');

let person = { name: "Bob", age: 30 };
let numbers = [1, 2, 3];

let output = `Name: ${name}, Type: ${typeof name}
Age: ${age}, Type: ${typeof age}
Is Student: ${isStudent}, Type: ${typeof isStudent}
Null: ${nothing}, Type: ${typeof nothing}
Undefined: ${notDefined}, Type: ${typeof notDefined}
BigInt: ${bigNumber}, Type: ${typeof bigNumber}
Symbol: ${sym.toString()}, Type: ${typeof sym}
Person: ${JSON.stringify(person)}, Type: ${typeof person}
Array: ${numbers}, Type: ${typeof numbers}`;

document.getElementById('output').textContent = output;

Explanation

  • let name = "Alice"; — Declares a string variable called name.
  • let age = 25; — Declares a number variable called age.
  • let isStudent = true; — Declares a boolean variable called isStudent.
  • let nothing = null; — nothing is explicitly set to null (no value).
  • let notDefined; — notDefined is declared but not assigned, so it is undefined.
  • let bigNumber = 12345678901234567890n; — bigNumber is a BigInt (large integer, note the n at the end).
  • let sym = Symbol('unique'); — sym is a Symbol, a unique and immutable primitive value.
  • let person = { name: "Bob", age: 30 }; — person is an object with properties name and age.
  • let numbers = [1, 2, 3]; — numbers is an array.
  • The output variable uses template literals and typeof to show each variable's value and type.
  • The result is displayed in the green output box above.
