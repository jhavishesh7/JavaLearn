Explanation ▼
let a = 10, b = 5;: Declares two variables a and b and assigns them values 10 and 5.
let output = ...: This is a template literal (enclosed in backticks) that allows embedding expressions like ${a + b}.
Addition: ${a + b}: Calculates the sum of a and b.
Subtraction: ${a - b}: Calculates the difference between a and b.
Multiplication: ${a * b}: Calculates the product of a and b.
Division: ${a / b}: Calculates the quotient of a and b.
Exponent: ${a ** b}: Calculates a raised to the power of b.
Modulus: ${a % b}: Calculates the remainder when a is divided by b.

Comparison:
Equal (==): ${a == '10'}: Checks if a is equal to the string "10".
Strict Equal (===): ${a === '10'}: Checks if a is strictly equal to the string "10".
Not Equal (!=): ${a != '10'}: Checks if a is not equal to the string "10".
Strict Not Equal (!==): ${a !== '10'}: Checks if a is not strictly equal to the string "10".
Greater Than: ${a > b}: Checks if a is greater than b.
Less Than: ${a < b}: Checks if a is less than b.

Logical:
AND: ${a > 5 && b < 10}: Checks if a is greater than 5 AND b is less than 10.
OR: ${a > 5 || b > 10}: Checks if a is greater than 5 OR b is greater than 10.
NOT: ${!(a > 5)}: Checks if a is NOT greater than 5.

Ternary:
${a > b ? 'a is greater' : 'b is greater'}: If a is greater than b, it prints "a is greater". Otherwise, it prints "b is greater".

// If-else

    output += "Adult\n";
}
    else {
      output += "Minor\n";

This block checks if age is 18 or older. If true, it appends "Adult" to output and a newline. If false, it appends "Minor" to output and a newline.

// Switch

    case "Monday":
      output += "Work day\n";

    case "Friday":

    break;

This block checks the value of day. If day is "Monday", it appends "Work day" to output and a newline. If day is "Friday", it appends "Almost weekend!" to output and a newline. For any other value, it appends "Regular day" to output and a newline.

5. Loops

let output = "For Loop:\n";: Initializes an empty string output with the text "For Loop:\n".
for (let i = 0; i < 3; i++) {
    output += i + " ";
}
This for loop iterates from 0 to 2 (3 times). For each iteration, it appends the current value of i followed by a space to output.

output += "\n\nWhile Loop:\n";: Appends a newline and the text "While Loop:\n" to output.
let j = 0;: Declares a variable j and initializes it to 0.
while (j < 3) {
    output += j + " ";
    j++;
}
This while loop continues as long as j is less than 3. For each iteration, it appends the current value of j followed by a space to output and increments j by 1.

    while (k < 3);
This do-while loop first appends the current value of k followed by a space to output and then increments k by 1. Then, it checks if k is less than 3. If true, it repeats the loop. This loop will execute at least once, then continue as long as k is less than 3.

output += "\n\nFor...of Loop:\n";: Appends a newline and the text "For...of Loop:\n" to output.
let fruits = ["Apple", "Banana", "Cherry"];: Declares an array fruits with three string elements.
for (let fruit of fruits) {
    output += fruit + " ";
}
This for...of loop iterates through each element in the fruits array. For each element, it appends the element followed by a space to output.

Explanation ▼
let numbers = [1, 2, 3];
This line declares an array numbers with three elements: 1, 2, and 3.

numbers.push(4);
This adds the number 4 to the end of the numbers array. Now it's [1, 2, 3, 4].

numbers.unshift(0);
This adds the number 0 to the beginning of the numbers array. Now it's [0, 1, 2, 3, 4].

let last = numbers.pop();
This removes the last element from the numbers array and stores it in the last variable. Now last is 4, and numbers is [0, 1, 2, 3].

let first = numbers.shift();
This removes the first element from the numbers array and stores it in the first variable. Now first is 0, and numbers is [1, 2, 3].

let doubled = numbers.map(n => n * 2);
This creates a new array doubled by applying a function to each element of numbers. The function n => n * 2 multiplies each number by 2. So doubled is [2, 4, 6].

let evens = numbers.filter(n => n % 2 === 0);
This creates a new array evens by filtering numbers. The function n => n % 2 === 0 checks if a number is even. So evens is [2].

let sum = numbers.reduce((total, n) => total + n, 0);
This calculates the sum of all elements in the numbers array. The reduce method takes a function and an initial value. The function (total, n) => total + n adds the current number n to the total. Starting with total = 0, it adds 1, then 2, then 3. So sum is 6.

9. Asynchronous JavaScript

// Callback
function fetchData(callback) {
    setTimeout(() => {
        callback("Data received");
    }, 1000);
}

// Promise
function fetchDataPromise() {
    return new Promise((resolve) => {
        setTimeout(() => {
            resolve("Promise resolved");
        }, 1000);
    });
}

// Async/Await
async function getData() {
    let data = await fetchDataPromise();
    return data;
}

let output = "Starting async operations...\n";

fetchData((data) => {
    output += `Callback: ${data}\n`;
    document.getElementById('output-9').textContent = output;
});

fetchDataPromise().then(data => {
    output += `Promise: ${data}\n`;
    document.getElementById('output-9').textContent = output;
});

getData().then(data => {
    output += `Async/Await: ${data}`;
    document.getElementById('output-9').textContent = output;
});

document.getElementById('output-9').textContent = output;
Explanation ▼
// Callback
function fetchData(callback) {
    setTimeout(() => {
        callback("Data received");
    }, 1000);
}
This is a callback function. It takes a callback function as an argument. It simulates an asynchronous operation (like fetching data) by waiting for 1 second and then calling the callback with the string "Data received".

// Promise
function fetchDataPromise() {
    return new Promise((resolve) => {
        setTimeout(() => {

        }, 1000);
    });

This is a Promise. It simulates an asynchronous operation that resolves after 1 second with the string "Promise resolved".

// Async/Await

    let data = await fetchDataPromise();

This is an async function. It uses the await keyword to wait for the fetchDataPromise() to resolve. It then returns the resolved data.

10. Error Handling

let output = "";

try {
    // This will throw an error
    let result = riskyOperation();
    output += result;
} catch (error) {
    output += `Caught an error: ${error.message}\n`;
} finally {
    output += "This always runs";
}

// Custom error
class CustomError extends Error {
    constructor(message) {
        super(message);
        this.name = "CustomError";
    }
}

try {
    throw new CustomError("Something went wrong");
} catch (error) {
    if (error instanceof CustomError) {
        output += `\nCustom Error: ${error.message}`;
    }
}

document.getElementById('output-10').textContent = output;
Explanation ▼
let output = "";
This line declares an empty string variable output.

try {
    // This will throw an error
    let result = riskyOperation();
    output += result;
}
This try block attempts to execute a function that throws an error. If it succeeds, it appends the result to output. If it fails, it moves to the catch block.

catch (error) {
    output += `Caught an error: ${error.message}\n`;
}
This catch block handles the error thrown by riskyOperation(). It appends the error message to output.

finally {
    output += "This always runs";
}
This finally block always executes, regardless of whether an error occurred or not. It appends the string "This always runs" to output.

// Custom error
class CustomError extends Error {
    constructor(message) {
        super(message);
        this.name = "CustomError";
    }
}
This is a custom error class that extends the built-in Error class. It takes a message parameter and assigns it to the error's message property and a specific name property.

try {
    throw new CustomError("Something went wrong");
}
This try block throws an instance of the CustomError class.

catch (error) {
    if (error instanceof CustomError) {
        output += `\nCustom Error: ${error.message}`;
    }
}
This catch block checks if the error is an instance of the CustomError class. If true, it appends the error message to output.

document.getElementById('output-10').textContent = output;: Displays the output string in the browser.

11. Modern Features

// Destructuring
let [x, y] = [1, 2];
let {name, age} = {name: "Alice", age: 25};

// Spread
let arr1 = [1, 2];
let arr2 = [...arr1, 3, 4];

// Template literals
let text = `Hello ${name}, you are ${age} years old`;

// Optional chaining
let user = { profile: { name: "Bob" } };
let city = user?.profile?.address?.city;

// Nullish coalescing
let setting = null;
let defaultValue = setting ?? "default";

let output = `Destructuring: x=${x}, y=${y}, name=${name}
Spread: ${arr2}
Template: ${text}
Optional Chaining: ${city}
Nullish Coalescing: ${defaultValue}`;

document.getElementById('output-11').textContent = output;
Explanation ▼
// Destructuring
let [x, y] = [1, 2];
This line uses array destructuring to assign the first element of the array [1, 2] to x and the second element to y. So x is 1 and y is 2.

let {name, age} = {name: "Alice", age: 25};
This line uses object destructuring to assign the name property of the object {name: "Alice", age: 25} to the name variable and the age property to the age variable. So name is "Alice" and age is 25.

// Spread
let arr1 = [1, 2];
let arr2 = [...arr1, 3, 4];
This line creates a new array arr2 by spreading the elements of arr1 (1, 2) and then adding 3 and 4. So arr2 is [1, 2, 3, 4].

// Template literals
let text = `Hello ${name}, you are ${age} years old`;
This line uses template literals to embed the values of name and age into the string. So text is "Hello Alice, you are 25 years old".

// Optional chaining
let user = { profile: { name: "Bob" } };
let city = user?.profile?.address?.city;
This line uses optional chaining to safely access the city property of user's profile's address. If any of these properties are null or undefined, it will return undefined instead of throwing an error. So city is undefined because user doesn't have an address property.

// Nullish coalescing
let setting = null;
let defaultValue = setting ?? "default";
This line uses the nullish coalescing operator ??. If setting is null or undefined, it returns "default". Otherwise, it returns the value of setting. So defaultValue is "default" because setting is null.

let output = ...: This is a template literal that embeds the results of these modern features.
document.getElementById('output-11').textContent = output;: Displays the output string in the browser.
