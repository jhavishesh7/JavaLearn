Web Technologies Explained

HTML ▼

HTML (HyperText Markup Language) is the standard language for creating web pages.

  • Defines the structure of web content using tags (e.g., <h1>, <p>, <img>).
  • Browsers read HTML to display text, images, links, forms, and more.
Example: 
<!DOCTYPE html>
<html>
  <head>
    <title>My Page</title>
  </head>
  <body>
    <h1>Hello, world!</h1>
    <p>This is a web page.</p>
  </body>
</html>

CSS ▼

CSS (Cascading Style Sheets) is used to style and layout web pages.

  • Controls colors, fonts, spacing, layout, and responsiveness.
  • Can be written inline, in the <style> tag, or in external files.
Example: 
body {
  background: #f5f7fa;
  color: #333;
  font-family: Arial, sans-serif;
}
h1 {
  color: #1976d2;
}

JavaScript ▼

JavaScript is the programming language of the web.

  • Makes web pages interactive (respond to clicks, show/hide content, fetch data, etc.).
  • Runs in the browser (and on servers with Node.js).
Example: 
document.getElementById('myBtn').onclick = function() {
  alert('Button clicked!');
};

HTTP & Web Servers ▼

HTTP (HyperText Transfer Protocol) is the protocol for transferring data on the web.

  • Browsers (clients) send requests to servers, which send responses back.
  • Common methods: GET (fetch data), POST (send data), PUT, DELETE.
Example HTTP Request: 
GET /index.html HTTP/1.1
Host: www.example.com

APIs ▼

API (Application Programming Interface) lets programs talk to each other.

  • Web APIs allow your site to fetch or send data to other services (like weather, maps, or your own backend).
  • APIs use HTTP and data formats like JSON or XML.
Example (fetching data with JavaScript): 
fetch('https://api.example.com/data')
  .then(response => response.json())
  .then(data => console.log(data));

Frontend & Backend ▼

Frontend is everything the user sees and interacts with (HTML, CSS, JavaScript in the browser).
Backend is the server-side logic, databases, and APIs that power the site.

  • Frontend: React, Angular, Vue, plain HTML/CSS/JS
  • Backend: Node.js, Python, Java, PHP, Ruby, etc.

Databases ▼

Databases store and organize data for web applications.

  • SQL Databases: Structured, use tables (e.g., MySQL, PostgreSQL, SQLite).
  • NoSQL Databases: Flexible, use documents or key-value pairs (e.g., MongoDB, Redis).
Example SQL Query: 
SELECT * FROM users WHERE age > 18;
